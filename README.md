# Community FAQ

Static site hosted on github pages that serves as an FAQ for our community.

## Develop (**osx only at the moment**)

- `brew install hugo`
- `git clone git@github.com:DataWallet/community.git`
- `cd community`
- `hugo server -D`

## Deploy
- `scripts/deploy.sh`
- That's it, really.
